记录一下使用git遇到的问题

Author: wangchang Date: August 28, 2019  15:34:58 Category: git
  • “error: Your local changes to the following files would be overwritten by merge”
    场景:同事提交a文件,自己在本地对a文件做了修改,修改完想 pull 时提示上面的错误
    解决一:
    第一步: git stash 先把你修改过的被追踪的文件和暂存的变更——并将它保存到一个未完结变更的堆栈中(随时可以重新应用)
    第二步: git pull 拉取代码到本地(此时不会报error)
    第三步: git stash pop 从git栈中获取到最近一次stash进去的内容，恢复工作区的内容。。获取之后，会删除栈中对应的stash。 (由于可能会stash多次，git使用栈管理，我们可以使用git stash list查看所有的stash , git stash clear: 清空Git栈。)
    第四步: 手动解决冲突
    第五步: commit + push
    解决二:
    第一步: add + commit
    第二步: pull 此时会提示自动合并失败,需要去手动合并
    第二步: 手动合并冲突
    第四步: add + commit + push
